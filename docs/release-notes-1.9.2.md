# Course Date Shift Pro 1.9.2

Course Date Shift Pro helps Moodle administrators reschedule course timelines with more control and confidence. This release line consolidates the guided review workflow, weekly workload insights, history and restore dashboard, and branded PDF reporting.

## Highlights

- Guided preview before applying real date changes
- `Simple review` and `Advanced review` modes
- Weekly course load overview with drill-down detail
- Editable recommended dates directly in the review table
- Linked date suggestions for assignment-style scheduling fields
- History dashboard with restore support
- Branded PDF export with before/after execution detail

## Included in This Release

- clearer GitHub-facing documentation and screenshot presentation
- curated screenshot set for release and publication
- stable PDF branding behavior for the current release line
- CI-validated repository structure aligned with Moodle Plugin CI
- rebuilt AMD assets included in `amd/build`
